## Descripción del Proyecto

El reto consiste en desarrollar un **Oracle Java Bot** que automatice las tareas del equipo de desarrollo de software y proporcione visibilidad tanto a los desarrolladores como al manager. Este bot se integra con Telegram como medio de comunicación seguro y cumple con los estándares de Oracle para servicios DevOps. 

El objetivo principal es aumentar la productividad y visibilidad de las actividades del equipo en un 20%, automatizando tareas como la gestión de To-do's y proporcionando acceso a la información en tiempo real, tanto para desarrolladores como para el manager.
